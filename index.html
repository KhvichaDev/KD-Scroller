Why KD Scroller?

Auto-Theming

Reads your card's background color and automatically generates perfectly contrasted button styles and hover states.

Grab & Drag

Native-feeling desktop grab-and-drag mechanics with smart cursor feedback and click-event protection.

Infinite Scroll

Built-in API for loading more items when reaching the end, complete with an animated loading spinner.

A11y & Keyboard

Full arrow-key navigation with smart form-element detection to prevent interference with inputs inside cards.

Zero Dependencies

Pure Vanilla JS. It even auto-injects its own CSS, meaning no extra files to configure in your bundler.

Smart Snapping

Choose between 'group' (viewport width paging) or 'item' (single element snapping) scroll modes.

Continuous Scroll

Long-pressing the navigation arrows triggers a smooth, continuous scrolling animation.

SPA Ready

A complete `destroy()` method ensuring memory-leak-free integration with React, Vue, or Next.js.

 Installation & Setup

                        
Install via npm or use via CDN. The library automatically injects its required CSS, so there are zero extra configurations needed.

                        
                        

                            # Install via npm

                            npm install kd-scroller


                            // Or use in Browser

                            <script src="https://unpkg.com/kd-scroller"></script>
                        

                        
                        

                            // HTML Structure

                            <div class="my-list">

                              <div class="card">Item 1</div>

                              <div class="card">Item 2</div>

                            </div>


                            // Initialization

                            import KDScroller from 'kd-scroller';


                            const scroller = new KDScroller('.my-list', { scrollMode: 'item' });
                        

                    


                    
                    

                        
 API Configuration Options

                        

                            

                                
                                    

                                        Option
                                        Type
                                        Default
                                        Description
                                    

                                
                                
                                    

                                        scrollMode
                                        String
                                        'group'
                                        'group' (viewport paging) or 'item' (single snap)
                                    

                                    

                                        gap
                                        Number | String
                                        null
                                        Override CSS gap (e.g., 20 or '1rem')
                                    

                                    

                                        itemWidth
                                        Number | String
                                        null
                                        Force fixed width on children elements
                                    

                                    

                                        btnTheme
                                        String
                                        'auto'
                                        'auto', 'light', 'dark', or hex/rgb color string
                                    

                                    

                                        continuousSpeed
                                        Number
                                        3
                                        Pixels per frame during long-press scroll
                                    

                                    

                                        hideArrowsOnTouch
                                        Boolean
                                        false
                                        Hides navigation arrows entirely on touch devices
                                    

                                    

                                        onReachEnd
                                        Function
                                        null
                                        Callback triggered when scrolled to end. Pass 'done()' or return Promise.
                                    

                                    

                                        leftIcon / rightIcon
                                        String
                                        SVG
                                        Raw SVG string to replace default arrows
                                    

                                
                            

                        

                    


                    
                    

                        

                            
 Methods

                            

                                scroller.destroy()
                                
Fully tears down the instance, removing all event listeners and injected DOM elements. Essential for Single Page Applications (SPA) to prevent memory leaks during route changes.
